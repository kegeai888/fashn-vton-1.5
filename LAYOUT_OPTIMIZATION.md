# WebUI 布局优化说明

## 🎨 优化内容

### 1. 容器宽度优化
- **修改前**: 90% 宽度，在大屏幕上过宽
- **修改后**: 最大 1600px，95% 宽度，适配电脑浏览器
- **响应式**: 
  - 1400px 以下: 最大 1200px
  - 1200px 以下: 100% 宽度

### 2. 左右列布局优化
- **修改前**: 左右各占 50% (scale=1:1)
- **修改后**: 左侧 45% / 右侧 55% (scale=5:6)
- **最小宽度**: 左侧 400px，右侧 500px
- **间距**: Row gap 24px，Column gap 16px

### 3. 组件高度优化
- **图片上传**: 从 400px 降至 280px
- **结果画廊**: 从 500px 增至 600px
- **最小高度**: 图片上传区域 300px

### 4. 间距优化
- **组件间距**: 统一为 12px
- **Markdown 标题**: 上 20px，下 12px
- **Accordion**: 上边距 16px
- **生成按钮**: 上边距 16px，宽度 100%

### 5. 样式细节
- **Tab 内边距**: 从 16px 增至 20px
- **卡片阴影**: 保持统一
- **圆角**: 统一 12px
- **按钮**: 全宽，渐变背景

## 📐 布局结构

```
Container (max-width: 1600px, 95%)
└── Row (gap: 24px)
    ├── Column Left (scale: 5, min-width: 400px)
    │   ├── 输入图像 (height: 280px × 2)
    │   ├── 基础设置
    │   ├── 高级参数 (Accordion)
    │   └── 生成按钮 (width: 100%)
    └── Column Right (scale: 6, min-width: 500px)
        ├── 结果画廊 (height: 600px)
        ├── 状态文本
        └── 保存路径
```

## 🖥️ 屏幕适配

| 屏幕宽度 | 容器宽度 | 布局 |
|---------|---------|------|
| > 1600px | 1600px | 左右分栏 |
| 1400-1600px | 1200px | 左右分栏 |
| 1200-1400px | 100% | 左右分栏 |
| < 1200px | 98% | 左右分栏 |

## ✅ 优化效果

- ✓ 减少左右空隙，更紧凑
- ✓ 组件宽度更合理
- ✓ 适配常见电脑屏幕（1920×1080, 2560×1440）
- ✓ 保持响应式设计
- ✓ 统一间距和样式
